# Class: ReflectiveShadowMap

Defined in: [babylonjs-source/packages/dev/core/src/Rendering/reflectiveShadowMap.pure.ts:34](https://github.com/BabylonJS/Babylon.js/blob/4d4dad2f008743af1de7c56c2cbe1a0c8879df33/packages/dev/core/src/Rendering/reflectiveShadowMap.pure.ts#L34)

Class used to generate the RSM (Reflective Shadow Map) textures for a given light.
The textures are: position (in world space), normal (in world space) and flux (light intensity)

### forceUpdateLightParameters

> **forceUpdateLightParameters**: `boolean` = `false`

Defined in: [babylonjs-source/packages/dev/core/src/Rendering/reflectiveShadowMap.pure.ts:101](https://github.com/BabylonJS/Babylon.js/blob/4d4dad2f008743af1de7c56c2cbe1a0c8879df33/packages/dev/core/src/Rendering/reflectiveShadowMap.pure.ts#L101)

Gets or sets a boolean indicating if the light parameters should be recomputed even if the light parameters (position, direction) did not change.
You should not set this value to true, except for debugging purpose (if you want to see changes from the inspector, for eg).
Instead, you should call updateLightParameters() explicitely at the right time (once the light parameters changed).
